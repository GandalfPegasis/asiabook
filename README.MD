# Asiabook Structure

1. index.js search for the route 
2. /routes/index -> go to the page you want e.g profile
3. /services/profile sanitize user requests
4. /controllers/profile (e.g getting profile data) -> call /dataaccess/profileDAO getProfile() function
5. go back to /services/profile convert data to defined structure (e.g object, arrays)
6. /services/profile render the data from service
7. mari 🗿
